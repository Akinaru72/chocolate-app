# 🍫 Simply Chocolate

Responsive landing page for a chocolate company built with HTML, CSS, and JavaScript.

## 🔗 Live Demo

[Live Page](https://akinaru72.github.io/chocolate-app/)

## 📖 About the Project

**Simply Chocolate** is a responsive promotional website designed to showcase a premium chocolate brand. The project includes adaptive layouts for mobile, tablet, and desktop devices, interactive navigation, a mobile menu, and a modal review form.

The website introduces visitors to the company's products, ingredients, manufacturing process, customer reviews, and contact information.

## ✨ Features

- Responsive design
- Mobile-first approach
- Burger menu for mobile devices
- Modal review form
- SVG sprite icons
- Retina image support (2x)
- Smooth navigation through page sections
- Interactive ingredient cards with overlay effects
- Newsletter subscription form
